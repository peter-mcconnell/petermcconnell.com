Escaparate

2022-11-29 :: :: 2 min read (356 words)

He estado alrededor por un tiempo y he hecho algunas cosas. Incluso algunas cosas buenas.

  • ayudó a llevar dos empresas a la adquisición
  • desarrolló un lenguaje de consulta personalizado (TQL, una variante opencypher) que salió al mercado [Java. Golang. ANTLR]
  • soporte agregado en el proveedor Terraform AWS para enviar sus propias claves a ACM [Ir]
  • desarrolló un marco de orquestación para investigadores en IA para programar cargas de trabajo [ Pitón. Terraformar. AWS. Local (Nvidia DGX). Qemu. Librería]
  • producto clair de redhat personalizado para que pueda usar fuentes internas para sus bases de datos de vulnerabilidad [Ir]
  • desarrolló una herramienta de cumplimiento que detecta paquetes para Python, Ruby, R, APT, Yum, APK y evalúa sus licencias [ Cascarón ]
  • Desarrolló un chatbot que utilizó una gran base de datos de foros de automóviles, el teorema de Bayes y más expresiones regulares de las que se pueden agitar para permitir que los usuarios hablen con los automóviles. ¡en 3 semanas! [PHP, Postgres]
  • creó una herramienta de seguridad “amald” que alerta en caso de que un punto final no tenga autenticación. disminución del tiempo de detección de las empresas para configuraciones incorrectas de autenticación [Ir]
  • Desarrollé un servicio de entrega de imágenes para una organización muy grande (30k+ ingenieros) [AWS, Python, Bash/Shell, OpenStack]
  • he trabajado en productos web para Google, Virgin, Red Bull, Disney y YouTube [ Pitón. PHP. Durante mi tiempo como líder en https://www.rehabagency.ai/work]
  • diseñar y construir a partir de varias infraestructuras de implementación [Kubernetes. Mesos. Maratón. Docker-componer. ]
  • desarrolló un complemento de Jira que extendió JQL (lenguaje de consulta de jira) para permitir la agrupación por “roles” [Java]
  • compitió en varios hackathons y code jams [Hacktoberfest, Ludum Dare, Google CTF]
  • fue mentor de muchos ingenieros increíbles a lo largo de los años, desde Junior hasta Staff
  • hablado en varias conferencias en el Reino Unido, Irlanda, Australia y los EE. UU. sobre docker, prometheus, bash, terraform, aws, etc.
  • cofundó una empresa de diseño web galardonada (31interactive)

20 años trabajando en tecnología y lo anterior son todas las “cosas buenas” en las que puedo pensar listado. Imagínate lo larga que debe ser la lista de “cosas malas”…
